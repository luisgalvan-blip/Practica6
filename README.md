Práctica 6. El objetivo fue aprender a manejar Git (ramas y commits) y conectar 
clases de lógica (como Carta) con clases gráficas (Canvas, Square, etc.) para dibujar 
una carta en una ventana.

Para que no le salgan errores de no se encuentra la clase, lo mejor es compilar todo 
de esta forma. Abra su terminal en la carpeta del proyecto y escriba:

rm *.class  
javac *.java
java Main
